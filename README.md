# Sort4Me

[Play Sort4Me on GitHub Pages!](https://chubarevgeniy.github.io/Sort4me/)

A simple React+TS mobile-friendly application to help you prioritize your thoughts, tasks, and problems.

## Purpose

Sort4Me is designed to resolve decision paralysis when you have a list of tasks, ideas, or options but don't know where to start. Instead of trying to rank everything at once, Sort4Me asks you to compare just two items at a time. Using a robust Merge Sort algorithm behind the scenes, it quickly determines your true priorities based on your simple A vs B choices.

## Usage

1. **Create a List**: Enter the items, tasks, or ideas you want to prioritize.
2. **Sort**: The app will present you with a series of pairs. Simply click the item that is more important or preferred.
3. **Review Results**: Once all comparisons are complete, you'll receive a perfectly ordered list of your priorities.
4. **Manual Adjustments**: If you change your mind, you can manually check off items or re-sort the list.
5. **Share**: Export your sorted list via a shareable link or copy it as plain text to send to others.
6. **History**: Your previous lists are automatically saved to your browser's local storage for easy access later.

## Features

- **Prioritize items**: Easily decide the order of importance using a simple pairwise comparison algorithm.
- **Manual Adjustments**: Make manual reorderings once the initial sort is completed.
- **Share**: Export your sorted list via a shareable link or copy as plain text.
- **History**: Keeps track of your previous lists using local storage.
- **Relaxing UI**: Designed with a calm, pastel color palette.

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Build for production: `npm run build`
